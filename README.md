# Healthcare Platform – Doctor Availability & Teleconsultation

This project is a full-stack healthcare web application that enables patients to easily find doctors, check availability, and consult them remotely using chat and video communication.

## 🚀 Features

* **Doctor Listing & Availability**

  * View available doctors
  * Status-based interaction

* **Appointment System**

  * Book consultations with doctors

* **Real-Time Chat**

  * Socket-based messaging between patient and doctor
  * Room-based chat sessions

* **Video Consultation (WebRTC)**

  * Peer-to-peer video calling
  * Room-based connection setup

* **Chatbot Assistant**

  * Floating chatbot on homepage
  * Provides quick guidance to users

* **User Roles**

  * Separate flows for Patients and Doctors

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express
* **Real-time:** Socket.io
* **Video:** WebRTC
* **Database:** MongoDB

## 🎯 Purpose

To improve accessibility to healthcare services by enabling remote consultations and reducing the time required to connect with doctors.
